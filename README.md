This is a stability calculator for Digital Linear Ion Trap programmed in Visual Basic, called “DAWI calculator for LIT”. This software can be used to compute a-q parameters at boundaries of the first stability region, and to predict the stability of ions with given mass-to-charge ratios (m/z) and parameters of the rectangular waveforms.

More details may be found in the paper:
Li Ding, Lulu Sun, Weimin Wang, The development of the digital ion trap, mass spectrometry reviews. 2026 
If you find this software useful, we ask that you please cite the paper. We welcome the submission of bugs, improvements, and suggestions to the software.
The software will be further enhanced to incorporate other configurations of ion traps.
